🌦️ Weather App (PyQt5):



A clean, desktop-based weather application built with PyQt5 that fetches real-time weather data using the OpenWeatherMap API.

🚀 Features:

🌍 Search weather by city name
🌡️ Displays real-time temperature (°F)
☁️ Weather condition description
😎 Emoji-based weather visualization
⚠️ Robust error handling (API + network)
🖥️ Simple and responsive PyQt5 GUI

🛠️ Tech Stack:

Python 3
PyQt5
Requests
OpenWeatherMap API

🎮 Usage:

Launch the app
Enter a city name
Click "Get Weather"
View:
Temperature
Weather description
Emoji indicator

⚠️ Error Handling
Handles:

Empty input
Invalid API key (401)
City not found (404)
Network issues
Server errors (5xx)

Errors are displayed directly in the UI.
